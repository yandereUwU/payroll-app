# Система расчета заработной платы

## Описание проекта

Данный проект представляет собой настольное приложение для автоматизации расчета заработной платы сотрудников. Приложение разработано с использованием платформы Electron и базы данных SQLite.

Система позволяет учитывать сотрудников, их должности, выполнять расчет заработной платы с учетом отработанных дней и дополнительных выплат, а также формировать отчеты.

## Функциональные возможности

- Авторизация пользователей с разграничением прав доступа
- Ролевая модель (администратор, бухгалтер, руководитель, сотрудник)
- Управление пользователями (создание, удаление, назначение ролей)
- Управление сотрудниками
- Управление должностями
- Расчет заработной платы
- Учет премий и налогов
- Просмотр истории начислений
- Формирование отчетов
- Экспорт данных в Excel

## Роли пользователей

### Администратор
- Полный доступ ко всем функциям системы
- Управление пользователями
- Управление сотрудниками и должностями
- Расчет заработной платы
- Просмотр отчетов

### Бухгалтер
- Работа с сотрудниками
- Расчет заработной платы
- Просмотр и формирование отчетов

### Руководитель
- Просмотр информации о сотрудниках
- Просмотр отчетов

### Сотрудник
- Просмотр собственной информации
- Просмотр начисленной заработной платы

## Технологии

- Electron
- Node.js
- SQLite3
- ExcelJS

## Установка и запуск

1. Установить Node.js (если не установлен)

2. Установить зависимости:


npm install


3. Запустить приложение:


npm start


## Особенности реализации

- Используется локальная база данных SQLite
- Реализована автоматическая инициализация таблиц при запуске
- Применена ролевая модель доступа
- Выполняется автоматический расчет налога (13%)
- Реализовано сохранение истории начислений

## Назначение

Программа предназначена для использования в небольших организациях, где требуется автоматизация расчета заработной платы сотрудников.
